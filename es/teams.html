🌍 Base de datos de las 48 selecciones

Base de datos de referencia: grupos · ranking FIFA · valor de plantilla · director técnico (nacionalidad / año de llegada / cargo anterior). Se usa como insumo para los análisis de cada partido.
📌 Nota: el ranking FIFA ya está completo para la mayoría de las selecciones — el top (primeros 6) viene de la lista publicada en 2026/4, y el resto del Top 50 completo publicado por ESPN en 2026/1 (incluye menciones honoríficas #58/#71); las selecciones fuera del Top 50 (Catar/Haití/Curazao/Nueva Zelanda/Cabo Verde/Arabia Saudita/Jordania/Uzbekistán/Ghana) quedan por ahora con "—". Los valores de plantilla provienen del listado de Transfermarkt; algunas selecciones siguen con "—" y el agente diario los completa de forma continua. La lista de directores técnicos está completa para las 48 selecciones. Todos los datos se obtienen mediante "búsqueda en línea + extracción de páginas web", sin depender de navegador.
